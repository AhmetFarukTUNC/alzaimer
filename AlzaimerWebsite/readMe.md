# Alzheimer Prediction Web App

This project is a web-based application built with ASP.NET Core MVC that allows users to upload brain MRI images and receive early Alzheimer disease predictions powered by AI.

## AI Model Classes

The AI model predicts the uploaded MRI image into one of the following four classes:

1. **Very Mild Demented**  
   - Early stage with very mild symptoms.  
   - Daily activities are largely unaffected.  

2. **Mild Demented**  
   - Mild cognitive impairment.  
   - Some difficulty in daily tasks.  

3. **Moderate Demented**  
   - Noticeable cognitive decline.  
   - Assistance may be needed for daily activities.  

4. **Non Demented**  
   - No signs of dementia.  
   - Cognitive functions are preserved and MRI appears normal.

## Features

- Upload brain MRI images for prediction.  
- Display prediction result and confidence percentage immediately on the same page.  
- Store predictions in the database associated with the logged-in user.  
- Simple and user-friendly interface built with Bootstrap.  

## Technologies

- ASP.NET Core MVC  
- Entity Framework Core  
- Flask + TensorFlow Keras AI Model  
- Bootstrap for UI  

## Usage

1. Register or log in to the web app.  
2. Upload a brain MRI image.  
3. View the predicted Alzheimer class and confidence percentage instantly.  
4. Predictions are saved in your account history.
